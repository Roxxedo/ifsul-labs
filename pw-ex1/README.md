# 📋 Aplicação de Lista com Filtros

Aplicação web simples construída utilizando HTML, CSS e JavaScript puro (DOM)

---

## 🧠 Sobre

Este projeto foi desenvolvido como um exercício durante o curso técnico no IFSul.

O objetivo era construir uma interface totalmente funcional utilizando o que a turma tinha aprendido até o momento, sendo apenas tecnologias base, sem uso de frameworks ou bibliotecas.

---

## ⚙️ Funcionalidades

- Renderização dinâmica da lista  
- Múltiplos filtros  
- Adição de novos itens via input  
- Atualização em tempo real via DOM  
- Interface simples e interativa  

---

## 🧩 Tecnologias

- HTML  
- CSS  
- JavaScript (manipulação de DOM)  

---

## 🚧 Observações

> ⚠️ Este projeto foi desenvolvido em contexto de aula e pode não refletir organização completa ou boas práticas de produção.

---

## 🎯 Foco

- Manipulação de DOM  
- Eventos (event handling)  
- Atualização dinâmica de interface  
- Gerenciamento de estado sem frameworks  

---

## ▶️ como executar

Abra o arquivo `index.html` no navegador.

Não há dependências externas.
