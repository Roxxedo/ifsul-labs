# 🌐 PW2 - Programação Web 2

Repositório contendo os exercícios da disciplina de Programação Web 2 (PW2).

---

## 🧠 Sobre

Este diretório reúne os exercícios desenvolvidos durante a disciplina de Programação Web 2.

Os exercícios estão organizados em pastas numeradas (`ex-01`, `ex-02`, etc.), seguindo a estrutura e orientações fornecidas pelo professor.

---

## ⚙️ Estrutura

A estrutura do projeto foi definida em aula e é compartilhada entre todos os exercícios:

- **ex-XX/** → diretórios dos exercícios  
- **source/** → contém todas as classes utilizadas nos exercícios  
- **autoload.php** → responsável pelo carregamento automático das classes  

Essa organização foi utilizada como base para introdução ao padrão **MVC (Model-View-Controller)** em PHP.

---

## 🔁 Compartilhamento de código

As pastas **source/** e o arquivo **autoload.php** são compartilhados entre todos os exercícios, servindo como base comum para reutilização das classes e organização do projeto.

---

## 🚧 Observações

> ⚠️ Os exercícios seguem o modelo proposto em aula, podendo variar em complexidade conforme o conteúdo apresentado no momento.
