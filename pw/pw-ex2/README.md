# 📦 Sistema de Inventário

Aplicação web simples para gerenciamento de itens, construída utilizando HTML, CSS e JavaScript puro (DOM)

---

## 🧠 Sobre

Este projeto foi desenvolvido como um exercício durante o curso técnico no IFSul.

O objetivo era construir um sistema funcional de inventário utilizando o que a turma tinha aprendido até o momento, utilizando apenas tecnologias base, sem uso de frameworks ou bibliotecas.

---

## ⚙️ Funcionalidades

- Cadastro de novos itens  
- Listagem dinâmica do inventário  
- Edição e remoção de itens  
- Atualização em tempo real via DOM  
- Organização simples dos dados  

---

## 🧩 Tecnologias

- HTML  
- CSS  
- JavaScript (manipulação de DOM)  

---

## 🚧 Observações

> ⚠️ Este projeto foi desenvolvido em contexto de aula e pode não refletir organização completa ou boas práticas de produção.

---

## 🎯 Foco

- Manipulação de DOM  
- Eventos (event handling)  
- Estruturação de dados simples  
- Atualização dinâmica de interface  

---

## ▶️ como executar

Abra o arquivo `index.html` no navegador.

Não há dependências externas.
