# 🐦 Protótipo de Rede Social

Aplicação web simples construída utilizando HTML, CSS e JavaScript puro (DOM), simulando uma pequena rede social com cadastro de usuários

---

## 🧠 Sobre

Este projeto foi desenvolvido como um exercício durante o curso técnico no IFSul.

O objetivo era construir um sistema funcional com cadastro de usuários, utilizando o que a turma tinha aprendido até o momento, com tecnologias base e sem uso de frameworks ou bibliotecas.

Para este exercício, optei por desenvolver um pequeno protótipo inspirado no Twitter.

---

## ⚙️ Funcionalidades

- Cadastro de usuários  
- Exibição dinâmica de publicações  
- Interface inspirada em uma rede social  
- Atualização em tempo real via DOM  
- Interação básica entre elementos da aplicação  

---

## 🧩 Tecnologias

- HTML  
- CSS  
- JavaScript (manipulação de DOM)  

---

## 🚧 Observações

> ⚠️ Este projeto foi desenvolvido em contexto de aula e pode não refletir organização completa ou boas práticas de produção.

---

## 🎯 Foco

- Manipulação de DOM  
- Eventos (event handling)  
- Estruturação de interface  
- Simulação de fluxo de usuários e publicações  

---

## ▶️ Como executar

Abra o arquivo `index.html` no navegador.

Não há dependências externas.
