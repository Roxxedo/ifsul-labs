# 🏭 Sistema de Cadastro de Equipamentos (POO)

Aplicação desenvolvida como exercício de Programação Orientada a Objetos, focada na modelagem de classes e manipulação de dados em memória.

---

## 🧠 Sobre

Este projeto foi desenvolvido durante o curso técnico no IFSul como um exercício de Programação Orientada a Objetos (POO).

A atividade fazia parte do processo de introdução aos princípios SOLID, com foco na separação de responsabilidades e organização do código.

O objetivo era construir uma estrutura simples utilizando classes, separando entidade, repositório e execução da aplicação.

---

## ⚙️ Estrutura

O sistema foi dividido em três partes principais:

- **Equipamento** → representa o objeto com seus atributos  
- **CadastroEquipamento** → responsável por gerenciar os registros (repositório)  
- **Main** → responsável por executar operações e testar o sistema  

---

## ⚙️ Funcionalidades

- Cadastro de novos equipamentos  
- Atualização de setor de um equipamento  
- Busca de equipamentos cadastrados  
- Remoção de equipamentos  
- Gerenciamento em memória (sem banco de dados)
